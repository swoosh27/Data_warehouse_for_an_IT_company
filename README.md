# Data_warehouse_for_an_IT_company

### Создание структуры базы данных:
1. **`Script_createDB_1.sql`** – создание таблиц измерений (Dimensions)
2. **`Script_createDB_2.sql`** – создание таблиц фактов (Facts)

### Наполнение данными:
3. **`Script_fillDB.sql`** – заполнение всех таблиц тестовыми данными

### Аналитические запросы:
4. **`Script_1.sql`** – анализ общего рабочего времени по проектам и пользователям с использованием `GROUPING SETS`
5. **`Script_2.sql`** – анализ количества задач по проектам и спринтам с использованием `CUBE`
6. **`Script_3.sql`** – анализ количества задач по проектам и приоритетам с использованием `ROLLUP`
7. **`Script_4.sql`** – анализ накопленного рабочего времени пользователей
8. **`Script_5.sql`** – рейтинг скорости выполнения проектов
